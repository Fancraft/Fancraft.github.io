Hexo 图片展示

Posted on 2024-07-29 In web

绝对路径

适合少量图片。图片放入 source/images 目录，每次 generate 会生成图片。通过类似于 ![](/images/image.jpg) 访问。

相对路径

将 config.yml 文件中的 post_asset_folder 选项设为 true 。 每次写新文章 hexo new [layout] <title> 就自动创建新文件夹，把图放入同命名文件夹，

图片引用：

{% asset_path slug %}
{% asset_img slug [title] %}
{% asset_link slug [title] %}

标签插件

结合 Typora 使用，将 Preferences 里的 images 设置为

可以直接复制粘贴图片到 Typora。

哥伦比亚概述

Posted on 2024-07-28 In Travel

行程

botoga → san agustin → popayan → cali → salento → filandia → medellin

概述

* 电压：除欧洲外，加拿大 / 中国 / 美国的充电设备都可以直接使用。

* 饮用水：大部分城市自来水可以直接饮用，小城镇可以跟本地人确认，或者买瓶装矿泉水。

* 防晒霜防蚊喷雾：必带，还有墨镜。

* 疫苗：虽然大部分人都打过，但旅行前还是建议打一针。另外，看彩虹河 Caño Cristales 进入国家公园 Serranìa de la Macarena National Park 需要疫苗证明。

* 气候：

  • 山里特别冷，加上下雨，又湿又凉。基本穿搭可以长 / 短袖 + 防水外套，有时候冷得加个秋衣秋裤也不为过。我没有带，所以早上一般是起来就打喷嚏流鼻涕，晒个太阳就好。
  • 大城市偏热，比如 Cali 和 Medellin，短袖短裤正舒适，只是会晒黑。

日全食

Posted on 2024-04-09 In Life

多云也要看日食

天气预报下午将会多云，偏偏日全食也在这个时候，不过去外面晒太阳躺着也行。窗外层层灰云没有散去的迹象，远处天地交界处倒是泛蓝。如果开车远离这里一两个小时，大概率会是晴天。月亮吞噬太阳一段时间后，对面楼竟然有了光阴。必须要下去了！天公作美。居民已经在有光的地方聚集。有人半开着驾驶座的门，侧身举着日食专用眼镜看了入神。老太太仰躺轮椅上欣赏。乌云似有似无地飘过，想要遮住太阳。可惜我重度近视和散光，用专用眼睛看出了好几个重影，月牙状橙黄的太阳慢慢变细。去掉日食眼镜，刺眼地白光就在我头上，我只能回避。肉眼盯着是看不到月亮的影子的，只瞎掉眼睛。

在月亮接近于完全覆盖住太阳时，天空整个暗了下来，路灯亮了，仿佛一下子进入黑夜。太阳只剩下一个圆环，还有一点点的偏差让它发出如戒指上的钻石般的光芒。于是，正式进入一分多钟的日全食。这时可以用肉眼观看。震撼的景象来了，整个太阳除了逃离出月亮的一环白光，整个芯都是暗黑色。周边的云和天都要比它亮。人群躁动起来了，抬头看罕见地日全食。

WiFi - 续

不装 Wifi 的弊端有三个：

  1. 由于外出使用 wifi 而导致大部分时间都不在屋里，猫猫有极大概率的饿肚子的风险。于是乎，回家时猫很兴奋地讨吃的。
  2. 也由于大部分时间在外面，冰箱里的食物不能及时吃掉而浪费。会有回家时太过疲惫而无法打包第二天的饭的情况。
  3. 在外餐饮消费频次增高。

总之，wifi 装回来了。可喜可贺！🎉

突然地回归网上冲浪生活有些不能自拔，也是个新生活的开始。

一切初始

Posted on 2024-03-24 In Life

一切初始

初衷

建立网站主要原因是简历上放个网站，原网站是纯 html 和 css 手写。在浏览互联网上留下足迹的大佬们后，决定使用别人搭建好的，自己修改即可。看了视频，参考了一些人的手记，顺便关注了感兴趣的并持续输出的博客博主。看到运行时间及长的博主都会被激励到，也希望能输出有用的文章。没有再更新的博主也有很多，他们的知识和思考都被储存在互联网上。互联网确实是一扇窗，让我看到想象不到的生活和人。

过程

尽管已有前人铺路，任何一个自己想要摸索的技能仍然很有挑战性。不论是科研还是学习一门语言，掌握整个系统的框架才能找到出错的原因，也才能更改和进步。

简单来说 Hexo 使用 javascript 来制造网络博客。如果你安装了 Hexo，那就只需要个主题 theme 和服务器。当然，对我来说，不想付钱买服务器就可以在 github 上部署。参考了太多博主，而一些教程已经落后，版本已更新不适用，所以我大概总结了遇到的麻烦，可供参考。

耗时大概一周才有个还算比较满意的博客，但还需要改正和添加的地方太多了。以后慢慢改，也是实行不需要一直在一个问题上栽跟头，适当地转移自己的注意有助于思考。（虽然我这几天一直是干坐好几个小时搞一个 bug 的状态 :3

当下

凑合着用。因为搞网站而把自己的其他事情搁置了，很专一，也很呆板。说不定以后直接链接 notion，毕竟整合也很方便。
