中国人对中国人情绪

Posted on 2024-08-19 In Life

已经抵达荷兰，慢慢整顿中。关于安置细节，可以暂时拖一拖。

今晚在微信群中看到转发的小红书帖子。一个博主喜欢分享在挪威的职业规划，老板某天跟她说有匿名人发邮件问公司福利这么好怎么运作的，指责和博主职业规划的帖子等等。邮件的意淫和扰乱博主工作的行为令人感到不适。互联网上，每人都要注重隐私，谨防不怀好意之人。好玩的是群里开始大骂中国人，例如 “防火防盗防同胞”“中国人只坑中国人”。此种同族裔之间的不信任感让我感到不适，从愤怒感到难过，到平和的情绪也花了点时间。

有此种说法的人，一是大多没有大局观（这话好熟悉，罗永浩？）。不光中国人坑中国人，但凡是出去看看，白人与白人，印度人与印度人，拉丁美人与拉丁美人…. 哪一个不是跟同族裔人斗智斗勇？没有同胞的欺骗，大帮子印度人为何都居住在一件小地下室？为什么印度老哥感到绝望，说加拿大人 suffer，刚带全家来没多久就要走？对于同族人的狠必也起于爱的深切。如若自己对任何人都平常心对待，只要看看，就知道缺心眼儿的人哪儿都有。

Hexo 图片展示

Posted on 2024-07-29 In web

绝对路径

适合少量图片。图片放入 source/images 目录，每次 generate 会生成图片。通过类似于 ![](/images/image.jpg) 访问。

相对路径

将 config.yml 文件中的 post_asset_folder 选项设为 true 。 每次写新文章 hexo new [layout] <title> 就自动创建新文件夹，把图放入同命名文件夹，

图片引用：

{% asset_path slug %}
{% asset_img slug [title] %}
{% asset_link slug [title] %}
